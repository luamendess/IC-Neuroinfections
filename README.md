> Projeto: Exploração de assinaturas de expressão gênica humana para melhorar a sensibilidade de testes metagenômicos no diagnóstico de neuroinfecções.

Este projeto propõe uma abordagem integrada, baseada em dados públicos, que combina mNGS e análise da expressão gênica humana para aprimorar o diagnóstico da meningite tuberculosa. 

A hipótese consiste na identificação e avaliação de assinaturas de expressão gênica capazes de discriminar casos de tuberculose meníngea, utilizando pipelines reprodutíveis e técnicas de aprendizado de máquina conforme a metodologia descrita na literatura. 

Adicionalmente, será avaliada a especificidade desses preditores em amostras de outras etiologias infecciosas do sistema nervoso central. 

Essa abordagem pode fornecer subsídios para futuras aplicações no aprimoramento do diagnóstico de neuroinfecções em ambientes clínicos.
